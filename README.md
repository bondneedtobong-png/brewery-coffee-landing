# Brewery Coffee — лендинг крафтового кафе

Одностраничный лендинг для спешелти-кафе в Москве. Тёмный минимализм с акцентами цвета крафта, многосекционная композиция со split-блоками, bento-сеткой галереи и встроенной формой бронирования. Полная адаптивность под десктоп / планшет / мобильный. Без фреймворков — только чистый HTML/CSS/JS.

## Демо

- [GitHub Pages](https://example.github.io/brewery_coffee_landing/) — *добавьте ссылку после деплоя*

## Что внутри

**Структура страницы** (по порядку прокрутки):
1. **Topbar** — анонс новой обжарки и режим работы
2. **Header** — лого с монограммой, нумерованная навигация, CTA «Забронировать стол»
3. **Hero** — split-композиция: слева заголовок с курсивным акцентом, факты-цифры, две кнопки; справа фото-карточка «Roast of the week» с круглой печатью «№1 по спешелти» и SCA-оценкой
4. **О нас** — большое фото зала с плашкой «7 лет», 4 фичи в ряд с круглыми иконками, блок статистики с анимированными счётчиками
5. **Меню** — табы (Эспрессо / Альтернатива / Десерты), 4-колоночная сетка карточек с фото, бейджами «Хит» / «Новинка», метками граммовки
6. **Цитата** — отзыв постоянного гостя с аватаром
7. **Атмосфера** — bento-галерея 6 фото с подписями на hover, главный кадр 2×2
8. **Бронирование** — split: слева текст + фичи + фото зала с круглой печатью-стампом; справа sticky-форма с валидацией
9. **Контакты** — стилизованная карта с пульсирующим пином и оверлеем + 4 карточки контактов
10. **CTA-баннер** — подписка на рассылку
11. **Footer** — расширенный 4-колоночный с навигацией, контактами, соцсетями и фоновой надписью

## Стек

- **HTML5** — семантическая разметка, ARIA для бургера, формы, иконок
- **CSS3** — Grid, Flexbox, кастомные свойства (60+ токенов), `clamp()` для fluid-typography, плавные анимации, media-queries
- **Vanilla JavaScript** — IIFE, без зависимостей. IntersectionObserver для reveal-анимаций и счётчиков; смена табов меню; маска телефона; валидация полей
- **Google Fonts** — Cormorant Garamond (заголовки, italic-акценты) + Manrope (текст, UI)
- **SVG inline** — все иконки и декоративные элементы вшиты в разметку
- **Unsplash** — фоновые и контентные изображения через CDN с параметрами оптимизации

## Подход к дизайну

- **Тёмный минимализм с тёплыми акцентами.** Палитра — глубокий тёмный фон (`#0E0C0A`) и тёплый сложный бежевый (`#D4B07A`).
- **Серифные италики как акцент.** Cormorant Garamond курсивом подчёркивает ключевые слова в заголовках — «в сердце», «ремесло», «задержаться».
- **Композиции, не списки.** Каждая секция использует split-сетку или bento-сетку — никаких однообразных «3 колонки иконок».
- **Декор на каждом блоке.** Бейджи, круглые печати, плашки на фото, пульсирующий пин, моргающий dot в topbar, бегущая строка под hero — поверхность не статична.
- **Анимации.** IntersectionObserver плавно проявляет блоки со staggered-задержкой, цифры статистики анимируются ease-out от 0 до целевого значения.
- **Fluid-typography.** Размеры заголовков через `clamp()` плавно масштабируются от мобильного к десктопу.
- **Доступность.** Семантические теги, `aria-expanded` на бургере, `aria-invalid` на полях с ошибкой, `prefers-reduced-motion` отключает все анимации, focus-visible подсветка.

## JavaScript-функциональность

- Бургер-меню на мобильных (открытие/закрытие, Esc, клик по ссылке, ресайз на десктоп)
- Плавный скролл к якорю с компенсацией высоты фиксированной шапки
- Шапка получает фон/блюр при скролле > 24px
- Reveal-анимации секций с safety-net (если IntersectionObserver не сработал — показываем через 2с)
- Анимированные счётчики статистики (ease-out cubic, 1.4с)
- Табы меню с переанимацией карточек
- Форма бронирования: маска `+7 (XXX) XXX-XX-XX`, валидация имени, телефона, даты (не в прошлом, не дальше 3 месяцев), времени, гостей; имитация отправки + сообщение успеха
- Подписка на рассылку: валидация email, фидбек на кнопке

## Адаптив

Брейкпоинты:

- **≥ 1200px** — десктоп, hero split 2 колонки, фичи 4 в ряд, меню 4 в ряд, контейнер до 1320px
- **960–1200px** — фичи и меню переходят в 2 колонки, hero ужимается
- **≤ 960px** — split-блоки выстраиваются в 1 колонку, hero card по центру, галерея в 2 колонки
- **≤ 768px** — бургер-меню, hero в 1 колонку, форма брони полноширинная
- **≤ 480px** — мобильные финальные подгонки: статистика и галерея в 1 колонку, CTA-форма в столбец

## Запуск

Никакой сборки — это статический сайт.

```bash
# Откройте index.html прямо в браузере:
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux

# Или поднимите локальный сервер (рекомендуется):
python -m http.server 8080
#    → http://localhost:8080

# Альтернативы:
npx serve .
php -S localhost:8080
```

## Деплой на GitHub Pages

```bash
git init
git add .
git commit -m "Initial commit: Brewery Coffee landing"
git branch -M main
git remote add origin https://github.com/<USERNAME>/brewery_coffee_landing.git
git push -u origin main
```

Далее: **Settings → Pages → Source: Deploy from a branch → main / root → Save**.
Через 1–2 минуты сайт будет доступен по `https://<USERNAME>.github.io/brewery_coffee_landing/`.

## Что можно доработать

- Заменить ссылки Unsplash на локальные оптимизированные WebP/AVIF

- Параллакс-эффект на hero-карточке при скролле
- Тёмная/светлая тема (сейчас только тёмная как часть концепции)
